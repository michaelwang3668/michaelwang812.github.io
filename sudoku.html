Sudoku Solver

Purpose

The purpose of this project was to create a Python program that solves a sudoku board using two methods: backtracking and forwardchecking.

Methods

In both methods, the way that the program is implemented is by placing numbers left to right, top to bottom.

The idea behind backtracking is checking if placing a number will result in seeing it more than once in the same row, column or box, then we know that this is not a solution, and backtrack.

The idea behind forward checking is that whenever we place a number, we add it to a list of constraints. In this case the constraint would be that the same number can not be in the same row, column, or box.

If the constraints are violated for a row, column, or box then we know that we made a mistake and know that we should not continue with this board.

Forward checking is more efficient than backtracking because it only places numbers that match the contraints and detects inconsistencies earlier thatn backtracking.
