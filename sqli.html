Blind Time-based SQLi

Purpose

The purpose of this project was to create a Python program that performs Blind Time-based SQL injection by performing different queries on a MySQL server.

Methods

The idea behind this SQLi is that you bruteforce one character at a time with the if, substring, and sleep functions in MySQL.

First you make a query and checks if the first character of the response is "A", then "B", and so on. If so, then sleep for 5 seconds. Now in the same manner we query for the second character in the response, and so on. Based on the time that it gets for the server to respond to us, we know if are correct or not.

Now that we have a strategy to make queries and retrieve the response, we first enumerate the databases, then the tables in the databases, and finally the columns in the tables.

Finally, now that we know the layout of the database, we can pull information from it with a SELECT statement and use the same method to bruteforce the response and get the data.
